## Installation

To clone and run this application, you'll need Git and Node.js (which comes with npm) installed on your computer.
From your command line:

```bash
  # Clone this repository
  $ git...

  # Go into the repository
  $ cd ...

  # Install dependencies
  $ npm install

  # Run the app
  $ npm run dev
```
